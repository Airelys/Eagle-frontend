¿Qué es Eagle?

Eagle es una aplicacion que busca resolver el problema de estimacion de parametros en modelos epidemiologicos, graficando tambien la solucion; y que ademas, da la posibilidad de resolver numericamente dicho modelo.

¿Qué modelos epidemiologicos pueden ser usados?

El software trabaja con prototipos de modelos epidemiologicos, entre ellos se pueden encontrar: SI, SIR, SIRS y SEIR. A continuacion se muestra donde se escoge el prototipo.
